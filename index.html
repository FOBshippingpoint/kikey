KikeyJS

Easy shortcut library for event handling, support key sequence, and shortcut recording.

Installation

Append following script into HTML head.


<script src="https://code.jquery.com/jquery-3.7.1.slim.min.js" integrity="sha256-kmHvs0B+OpCW5GVHUNjv9rOmY0IvSIRcf7zGUDTDQM8=" crossorigin="anonymous"></script>

Or via npm package manager.


npm install kikey

Now, Press G

Here, we instantiated a kikey object by calling Kikey() function. Register key bindings with kikey.on(), and remove the callback by kikey.off(). The key binding should always in lowercase, except for modifiers.


// Initialize kikey
const kikey = Kikey();

function typeGettingStarted() {
  // Remove event handler
  kikey.off(typeGettingStarted);
  startTyping();
}
// Bind event handler for key `g`, notice that `shift + g` would not work
kikey.on("g", typeGettingStarted);

Add ModifiersCtrl + Alt + U

We may want to add modifiers like Ctrl + S for saving file etc. In kikey, modifiers were represented in single uppercase letter:

  • C: Ctrl
  • S: Shift
  • A: Alt
  • M: Meta (Macintosh's Command Key or Windows Key)
Each key binding should concat with - (dash) for example, A for Alt, S-o for Shift + O, and C-A-e for Ctrl + Alt + e. The order of modifiers doesn't matter, but the main key should always place at rear.

        

// Listen to key `Ctrl+Alt+M`, mind the case
kikey.on("C-A-m", () => {
  // Start or stop the animation
  $("#letter-M").classList.toggle('rotate');
});
      

        
Key Sequence

        
Try press Shift + S → E → Q.

        

// Listen to key `Ctrl+Alt+U`
kikey.on("S-s e q", () => {
  $("#letter-M").classList.toggle('rotate');
});
